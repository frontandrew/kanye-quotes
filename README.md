'KanyeQuotes' is an app for showing quotes from Kanye West's Twitter feed.

### Features:

- Show quotes from Kanye West's Twitter feed;
- Show next quot every 15 sec automatically;
- User can click 'Next post' and see next.

### Use technology:

- ECMAScript 2019;
- Fetch API;
- CSS;
- HTML5;
- BEM methodology.
